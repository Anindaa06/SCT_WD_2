## 📌 About
An interactive and user-friendly Stopwatch Web Application built 
with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

## ✨ Features
- ▶️ Start, Pause, and Reset functionality
- ⏱️ Accurate time tracking in HH:MM:SS:ms format
- 🏁 Lap time recording and display
- 📋 Scrollable lap history list
- 🎨 Clean dark-themed modern UI
- 📱 Fully responsive on all screen sizes
- ⚡ Lightweight — single index.html file

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (setInterval API)
